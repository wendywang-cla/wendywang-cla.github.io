作用：CSS 用于设置网页的样式及布局——比如，可以更改内容的字体、颜色、大小以及间距，或是将其分列，或是添加动画及赋予内容其它装饰性的特征。

最后，CSS 如何选择要改变样式的HTML标签呢？

在讲选择器前，需要了解一个CSS中最基本最核心的一个概念。

CSS 盒模型

可以认为每个html标签都是一个方块，然后这个方块又包着几个小方块，如同盒子一层层的包裹着，这就是所谓的盒模型。 – 掘金

盒模型包含一下概念：width height padding border margin 千万注意顺序，还有一个神奇的IE盒模型（这是坑，面试也常问）

注意：行内元素属性有不同（width height margin padding 都不同）

CSS 选择器

选择器简单的说就是“找到” HTML 标签的方法

基本可以分成四大类

  1. 基本选择器

  2. 关系选择器

  3. 属性选择器

  4. 伪类选择器

四大基本选择器

通配符、class、id、标签

通配符 *

在CSS中,一个星号(*)就是一个通配选择器.它可以匹配任意类型的 HTML 元素

一般用来清除浏览器默认样式，但是每个浏览器默认样式不同，为了确保样式不会因为浏览器的不同而引起差异，所以在某些项目中会清除浏览器默认样式。

class 选择器

首先在 HTML 开始标签内设置 class属性，起一个属性名（最好能表示标签含义，不要使用无意义的名字命名，例如 a 1 ），之后，用 . + ‘属性名’，上图的是 ‘ .box ’，如此一来就能选中 HTML 中设置好 class 为 ‘ box ’ 的 HTML 元素。

思考：当多个 HTML 元素都设置为同一class名，能否都选中？

ID 选择器

依据 class 选择器的描述，可以推测出：id 选择器的属性名称为 id ，选择器的符号为：’ # ‘

tips：id在同一 HTML 文档中，只能唯一
